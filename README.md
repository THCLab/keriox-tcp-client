# keriox-tcp-client

### Usage

    cargo run -- [FLAGS] [OPTIONS]
```
FLAGS:
    -E               run eve demo (default)
    -B               run bob demo
    -S               run sam demo

OPTIONS:
    -H <host>        hostname on which we would listen, default: localhost
    -P <port>        port on which we would open TCP connections, default: 5621
```
